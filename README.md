# Villa de Leyva AI Heritage Guide

An AI-powered multi-agent system that transforms heritage tourism
in Villa de Leyva, Colo=mbia through contextual narratives,
personalized recommendations, and augmented reality visualization.

## Architecture
Tourist (Unity AR) ←→ Azure Functions ←→ Microsoft Agent Framework
├── Heritage Narrator Agent
├── Tourism Recommender Agent
└── Hotelier Assistant Agent
↕
Azure MCP Servers
Azure OpenAI (GPT-4o)
Azure AI Search (RAG)
Azure Cosmos DB

## Tech Stack
•⁠  ⁠*AI Platform*: Microsoft Foundry, Microsoft Agent Framework
•⁠  ⁠*Cloud*: Azure Functions, Azure OpenAI, Azure Cosmos DB, Azure AI Search
•⁠  ⁠*MCP*: Azure MCP Servers for heritage data access
•⁠  ⁠*Frontend*: Unity 6 with AR Foundation, TextMeshPro
•⁠  ⁠*Data Pipeline*: R (tidyverse) → JSON → Azure → Unity
•⁠  ⁠*Dev Tools*: VS Code, GitHub Copilot Agent Mode
